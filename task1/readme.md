# Creditworthiness Prediction using Machine Learning

## Overview

This project predicts whether a customer is creditworthy using historical financial information.

The objective is to help banks and financial institutions automate loan approval decisions by classifying applicants into:

- Good Credit(low risk)
- Bad Credit(high risk)

--

## Dataset

person_age
person_income
person_home_ownership
person_emp_length
loan_intent
loan_grade
loan_amnt
loan_int_rate
loan_percent_income
cb_person_default_on_file
cb_person_cred_hist_length

Target

loan_status

- 1 = non-default
- 0 = default

---

## Machine Learning Algorithms

- Logistic Regression
- Decision Tree
- Random Forest

---

## Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC Score
